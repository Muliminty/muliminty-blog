## 标题

使用井号（`#`）加空格来创建标题。井号的数量表示标题的级别。

<Callout>
  在文章中使用 `h2` 标签而不是 `h1` 标签。单个页面上过多的 `h1` 标签可能会影响 SEO。文章标题已经作为 `h1`。
</Callout>

```md
# 标题 1

## 标题 2

### 标题 3

#### 标题 4

##### 标题 5

###### 标题 6
```

<h1>标题 1</h1>
<h2>标题 2</h2>
<h3>标题 3</h3>
<h4>标题 4</h4>
<h5>标题 5</h5>
<h6>标题 6</h6>

## 段落

要创建段落，使用空行分隔一行或多行文本。

{/* prettier-ignore */}
```md
<!-- 空行 -->
我爱《星球大战》。
<!-- 空行 -->
我最喜欢的是《星球大战III：西斯的复仇》。
<!-- 空行 -->
```

我爱《星球大战》。

我最喜欢的是《星球大战III：西斯的复仇》。

---

## 斜体

使用一个星号（`*`）或下划线（`_`）来使文本变为斜体。

{/* prettier-ignore */}
```md
我 *爱* 《星球大战》。
我最 _喜欢_ 的是《星球大战III：西斯的复仇》。
```

我 _爱_ 《星球大战》。
我最 _喜欢_ 的是《星球大战III：西斯的复仇》。

---

## 加粗

使用两个星号（`**`）或下划线（`__`）来使文本加粗。

{/* prettier-ignore */}
```md
我 **爱** 《星球大战》。
我最 __喜欢__ 的是《星球大战III：西斯的复仇》。
```

我 **爱** 《星球大战》。
我最 **喜欢** 的是《星球大战III：西斯的复仇》。

---

## 斜体加粗

使用三个星号（`***`）或下划线（`___`）来使文本同时加粗和斜体。

{/* prettier-ignore */}
```md
我 ***爱*** 《星球大战》。
我最 ___喜欢___ 的是《星球大战III：西斯的复仇》。
```

我 **_爱_** 《星球大战》。
我最 **_喜欢_** 的是《星球大战III：西斯的复仇》。

---

## 水平分割线

使用三个连字符（`---`）、星号（`***`）或下划线（`___`）来创建水平分割线。

{/* prettier-ignore */}
```md
<!-- 空行 -->
---
<!-- 空行 -->
***
<!-- 空行 -->
___
<!-- 空行 -->
```

## 链接

要创建链接，将链接文本放在方括号（`[]`）中，然后紧跟着将 URL 放在圆括号（`()`）中。

```md
Micro 是 [astro-nano](https://github.com/markhorn-dev/astro-nano) 的一个分支。
```

Micro 是 [astro-nano](https://github.com/markhorn-dev/astro-nano) 的一个分支。

---

## 有序列表

要创建有序列表，添加带有数字和句点的行项目。使用缩进来创建嵌套列表。

```md
1. 项目 1
2. 项目 2
   1. 子项目 1
   2. 子项目 2
3. 项目 3
```

1. 项目 1
2. 项目 2
   1. 子项目 1
   2. 子项目 2
3. 项目 3

---

## 无序列表

要创建无序列表，在行项目前添加连字符（`-`）、星号（`*`）或加号（`+`）。不要混用。使用缩进来创建嵌套列表。

```md
- 项目 1
- 项目 2
  - 子项目 1
  - 子项目 2
- 项目 3
```

- 项目 1
- 项目 2
  - 子项目 1
  - 子项目 2
- 项目 3

---

## 图片

要添加图片，添加一个感叹号（`!`），然后在方括号（`[]`）中添加替代文本，最后在圆括号（`()`）中添加图片的路径或 URL。

```md
![替代文本](图片路径)
```

### 相对路径

使用 `![替代文本](./图片.*)` 模式，相对于 Markdown 文件所在的文件夹。注意句点。

```md
![X翼星际战斗机](./x-wing.jpeg)
```

![X翼星际战斗机](./x-wing.jpeg)

### 公共图片

使用 `![替代文本](/图片.*)` 模式，相对于 `public` 文件夹。没有句点。

```md
![Y翼星际战斗机](/y-wing.jpeg)
```

![Y翼星际战斗机](/y-wing.jpeg)

### 外部图片

使用 `![替代文本](URL)` 模式。

```md
![纳布N-1星际战斗机](https://raw.githubusercontent.com/trevortylerlee/n1/main/n1.jpeg)
```

![纳布N-1星际战斗机](https://raw.githubusercontent.com/trevortylerlee/n1/main/n1.jpeg)

## 引用

要添加引用，在段落前添加大于号（`>`）。对于多行引用，在每一行前添加大于号，并包含一个空行。

```md
> 这就是自由如何消亡的...
>
> 伴随着雷鸣般的掌声。
```

> 这就是自由如何消亡的...
>
> 伴随着雷鸣般的掌声。

---

## 删除线

使用波浪号（`~`）符号来添加删除线。

```md
~我不喜欢沙子。~ 它粗糙、刺激，让人不舒服。
```

~我不喜欢沙子。~ 它粗糙、刺激，让人不舒服。

---

## 下标

使用 `<sub>` 标签来表示下标。

```md
H<sub>2</sub>O
```

H<sub>2</sub>O

---

## 上标

使用 `<sup>` 标签来表示上标。

```md
E=mc<sup>2</sup>
```

E=mc<sup>2</sup>

---

## 键盘按键

使用 `<kbd>` 标签来表示键盘上的按键。

```md
<kbd>CTRL</kbd> + <kbd>ALT</kbd> + <kbd>Delete</kbd>
```

<kbd>CTRL</kbd> + <kbd>ALT</kbd> + <kbd>Delete</kbd>

---

## 缩写

使用 `<abbr>` 标签来表示缩写。

```md
<abbr title="图形交换格式">GIF</abbr>
```

<abbr title="图形交换格式">GIF</abbr>

---

## 高亮

使用 `<mark>` 标签来表示高亮文本。

```md
做或不做。<mark>没有尝试。</mark>
```

做或不做。<mark>没有尝试。</mark>

## 任务列表

将列表与方括号（`[]`）结合使用，表示复选框。在方括号内输入 `x` 表示任务已完成。

```md
- [x] 建造光剑
- [ ] 通过绝地试炼
- [ ] 训练学徒
```

- [x] 建造光剑
- [ ] 通过绝地试炼
- [ ] 训练学徒

---

## 表格

使用三个或更多连字符（`-`）作为列标题，并使用竖线（`|`）分隔每一列。可以通过在连字符的左侧、右侧或两侧添加冒号（`:`）来对齐列中的文本。

```md
| 项目   | 数量 |
| :----- | ----: |
| X翼    |     1 |
| Y翼    |     2 |
| A翼    |     3 |
```

| 项目   | 数量 |
| :----- | ----: |
| X翼    |     1 |
| Y翼    |     2 |
| A翼    |     3 |

## 脚注

添加一个插入符号（`^`）和方括号内的标识符（`[^1]`）。标识符可以是数字或单词，但不能包含空格或制表符。

```md
这是一个脚注，[^1] 这是另一个。[^2]
[^1]: 这是第一个脚注。
[^2]: 这是第二个脚注。
```

这是一个脚注，[^1] 这是另一个。[^2]
[^1]: 这是第一个脚注。
[^2]: 这是第二个脚注。

参见页面底部查看脚注。

## 代码

要将单词或短语表示为代码，请将其包裹在反引号（`` ` ``）中。

```md
`package.json`
```

`package.json`

---

## 代码块

通过将一段有效代码包裹在三个反引号（`` ``` ``）中来表示代码块。要启用语法高亮，请在第一个三个反引号后键入语言的简写符号。[简写符号参考](https://shiki.style/languages)。

````
```js
  function hello() {
    console.log("hello world");
  }
```
````

```js
function hello() {
  console.log("hello world");
}
```
