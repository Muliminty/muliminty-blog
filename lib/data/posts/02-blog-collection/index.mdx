## 使用 `blog` 集合：

`blog` 集合位于 `src/content/blog` 目录下。

```
📁 /src/content/blog
└── 📁 post-1
      └── 📄 index.md
└── 📁 post-2
      └── 📄 index.mdx
```

在上面的示例中，将会生成两篇博客文章，文件夹名称代表文章的 slug。

- `https://example.com/blog/post-1`
- `https://example.com/blog/post-2`

---

## 提供元数据

每篇文章都需要提供元数据。

```astro
---
title: "Blog Collection";
description: "How to add posts to the blog.";
date: "2024-03-21";
draft: false;
---
```

| 字段         | 必填 | 类型     | 备注                                             |
| :----------- | :--- | :------- | :----------------------------------------------- |
| title        | 是   | string   | 文章的标题。用于 SEO 和 RSS。                    |
| description  | 是   | string   | 文章的描述。用于 SEO 和 RSS。                    |
| date         | 是   | string   | 必须是有效的日期字符串（能够被解析）。           |
| draft        | 否   | boolean  | 如果 draft: true，内容将不会被发布。             |

---

剩下的就是在元数据下方编写内容。

```astro
---
title: "Blog Collection";
description: "How to add posts to the blog.";
date: "2024-03-21";
draft: false;
---
