## npm 命令

| 命令                   | 操作                                               |
| :--------------------- | :------------------------------------------------- |
| `npm install`          | 安装依赖                                           |
| `npm run dev`          | 启动本地开发服务器，地址为 `localhost:3000`        |
| `npm run build`        | `rm -rf .next && next build`                       |
| `npm run start`        | `next start`                                       |
| `npm run lint`         | `next lint`                                        |
